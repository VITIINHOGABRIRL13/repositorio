Bom, começamos pelo documento html é uma Linguagem de Marcação de HiperTexto é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para disso fomos para o http, o http é um protocolo de comunicação utilizado para sistemas de informatica de hipermídia, ele funciona como um sistema de solicitação - resposta, por exemplo quando um navegador web faz uma solicitação a um servidor web. Após isso, teve os subdomínios que sua função principal seria organizar e agrupar recursos relacionados em uma seção separada de um domínio. Os domínios seriam os endereços dos sites, por exemplo o google sendo https(protocolo) email(subdomínio) google(domínio) .com(domínio secundário). Também vimos sobre os métodos GET e POST, o método GET é um dos métodos do http(Hipertext Transfer protocol) utilizado para requisitos e recursos em um servidor, já o POST também é um dos métodos http, mas é utilizado para enviar informações para o servidor.
